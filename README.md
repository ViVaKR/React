# 시작 (Start)

[![node](https://nodejs.org/static/images/logo.svg)](https://nodejs.org/en)

```bash
    $ npx create-react-app my-app
    $ cd my-app
    $ npm start  
    # browser => http://localhost:3000,http://ip: 3000, # 3000 번 포트 충돌시 조치
    # 기타 사용 가능한 명령어들 : $ npm run build, $ npm test, $ npm run eject
```
